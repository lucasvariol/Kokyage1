# Images Blog

Placer ici les images des articles de blog.

## Format recommandé

- **Dimensions** : 1200×630px (format OpenGraph pour réseaux sociaux)
- **Format** : JPG (meilleure compatibilité) ou WebP (meilleure compression)
- **Poids** : < 200 KB optimisé
- **Nommage** : slug-article.jpg (ex: `sous-location-legale.jpg`)

## Utilisation

Dans le frontmatter de l'article markdown :

```markdown
---
image: "/images/blog/mon-article.jpg"
---
```

## Images actuelles

- `default.jpg` - Image par défaut si aucune image spécifiée
- `sous-location-legale.jpg` - Article "Comment sous-louer son logement légalement"

## Outils de création d'images

### Gratuits
- **Canva** : Modèles prêts 1200×630px
- **Unsplash** : Photos libres de droits haute qualité
- **Remove.bg** : Supprimer les fonds d'images

### Optimisation
- **TinyPNG** : Compression JPG/PNG sans perte de qualité
- **Squoosh** : Convertir en WebP
- **ImageOptim** : Optimisation batch

## Template Canva recommandé

1. Fond dégradé violet (#667eea → #764ba2)
2. Titre de l'article en gros (bold, blanc)
3. Logo Kokyage en bas à droite
4. Icône/illustration liée au sujet
